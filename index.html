Architecture

Code of Convolution Neural Network


    model=keras.Sequential([
    keras.layers.Conv2D(filters=32,kernel_size=3,activation="relu",input_shape=(100,100,1)),
    keras.layers.Conv2D(filters=64,kernel_size=3,activation="relu"),
    
    keras.layers.MaxPooling2D((4,4),padding='same'),
    keras.layers.Dropout(0.2),
    
    keras.layers.Conv2D(filters=128,kernel_size=3,activation="relu"),
    keras.layers.Conv2D(filters=128,kernel_size=3,activation="relu"),
    keras.layers.MaxPooling2D((4,4),padding='same'),
    keras.layers.Dropout(0.2),
    
    keras.layers.Conv2D(filters=256,kernel_size=3,activation="relu"),
    keras.layers.MaxPooling2D((4,4),padding='same'),
    keras.layers.Dropout(0.2),
    
    keras.layers.Flatten(),
    keras.layers.Dense(128,activation="relu"),
    keras.layers.Dense(64,activation="relu"),
    keras.layers.Dense(30,activation="softmax")
])
